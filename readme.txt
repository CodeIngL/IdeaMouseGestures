IdeaMouseGestures - IntelliJ IDEA plugin for handling mouse gestures.
=====================================================
version 1.10

Information
-----------
To build the project you should copy openapi.jar, jdom.jar and annotations.jar
from IDEA's lib directory to the 'lib' directory.


Installation
-----------
Copy IdeaMouseGestures.jar to the IDEA_HOME/config/plugins directory.


Product Home Page:

   http://www.smardec.com/products/idea.html

Support:

   idea@smardec.com

Depends on MouseGestures - open source pure Java library for recognition and
processing mouse gestures (http://www.smardec.com/products/mouse.html).
